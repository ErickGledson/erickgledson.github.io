O que é Mongodb?
O Mongodb é um banco de dados NoSQL, orientado a documentos e open source. O modo em que ele armazena os dados é muito interessante, pois é semelhante ao JSON: { key: value }.

Instalação do Mongodb no Windows
Eu tenho o Windows 10 na minha máquina, espero que os procedimentos feitos funcionem em todas as versões do Windows.

Passo 1
Faça o download do Mongodb. As versões de 64 bits do Mongo estavam rodando apenas em Windows Server 2008 R2, Windows 7 e as novas versões (8, 8.1 e 10) do Windows 64 bits. Já as versões de 32 bits rodam em Windows Vista (e superior: 7, 8, 8.1 e 10) e também em sistemas mais antigos, tal como o XP.

Passo 2
Após o download do .msi, dê dois cliques e faça a instalação padrão (next, next, next, finish).

Passo 3
O Mongo requer uma pasta para salvar dados, como default é \data\db. Crie a pasta no seu C:, ficando C:\data\db. Depois, configure a variável de sistema, colocando o caminho de instalação do Mongo (que no meu PC é C:\Program Files\MongoDB\Server\3.2\bin) na variável "Path".

Passo 4
Assim que todas as configurações forem feitas, abra o prompt de comando do Windows e digite "mongod" (sem aspas). Ao carregar, abra um novo prompt de comando e digite "mongo".

Conclusão
Pronto! Você tem uma instalação do Mongodb em sua máquina, pronta e funcional!
